# OptimalFlow Logistics — Supply Chain Correlation Analysis

This repository contains the correlation analysis for supplier performance metrics
as part of OptimalFlow Logistics' data-driven supply chain optimization project.

**Email:** 23f1002293@ds.study.iitm.ac.in

## Files Included

- **correlation.csv** — Exported correlation matrix computed using Excel Data Analysis ToolPak  
- **heatmap.png** — Excel correlation heatmap using Red–White–Green conditional formatting  
- **README.md** — Documentation and validation information  

## Method Summary

1. Enabled *Analysis ToolPak* in Excel  
2. Used **Data → Data Analysis → Correlation**  
3. Selected all 5 supply chain KPI columns  
4. Generated matrix and applied **conditional formatting**  
   - Red = low correlation  
   - White = medium  
   - Green = high  
5. Exported the matrix as CSV  
6. Captured heatmap screenshot (within 400×400 to 512×512 pixels)
